mspac

An apt-like wrapper for pacman.

Getting Started

In order to run this software, you need Python, it does not matter what version (provided it is updated enough), mspac will run in Python 2 and Python 3.

Installation

You can use pip in order to retrieve the package from PyPI: sudo pip install mspac

Usage

mspac [-h] [operation [packages ...]]

Use `mspac -h` to get a complete list of available options. The availabe ones at the time of writing are:

  • sync: Updates databases and upgrades packages. No arguments.
    mspac sync
  • autoremove: Removes orphaned packages. No arguments.
    mspac autoremove
  • update: Updates package databases. No arguments.
    mspac update
  • upgrade: Upgrades all pending packages. No arguments.
    mspac upgrade
  • install: Installs given packages. Arguments: a list of packages.
    mspac install package1 [package2 [package...]]
  • remove: Removes given packages. Arguments: a list of packages.
    mspac remove package1 [package2 [package...]]
  • show: Shows detailed info about given packages. Arguments: a list of packages.
    mspac show package1 [package2 [package...]]
  • list: Shows packages matching the given keywords. Arguments: a list of keywords.
    mspac list keyword1 [keyword2 [keyword...]]

Examples

Installing a package

The following command will install package1 and package2.

mspac install package1 package2

Removing a package

You can also remove package1 and package2.

mspac remove package1 package2

Updating the repositories

To update the databases of repositories:

mspac update

Upgrading packages

Upgrades all packages that are not up-to-date:

mspac upgrade

Updating repositories and upgrading packages

Updates the repositories and then upgrades all packages that are not up-to-date

mspac sync

Remove unused / orphan packages

Will remove packages that are no longer needed in the system:

mspac autoremove
